# Kündigung formulieren

## Prompt
```
Rolle: Arbeitsrechtlich versierter Kommunikationsexperte
Kontext: Ich möchte meine Stelle bei [FIRMA] kündigen
Meine Position: [JOBTITEL]
Dauer der Beschäftigung: [ZEIT]
Grund der Kündigung: [NEUER JOB/UMZUG/UNZUFRIEDENHEIT/PRIVAT]
Verhältnis zum Arbeitgeber: [GUT/SPANNUNGSREICH/NEUTRAL]
Aufgabe: Formuliere eine professionelle Kündigung
Einschränkungen:
- Rechtlich korrekt (Fristen, Formvorschriften)
- Brückenbauer, nicht Brückenabbrecher
- Keine Details zum neuen Arbeitgeber nennen
Ausgabe: Kündigungsschreiben mit korrekter Frist, Gesprächseinstieg für persönliche Übergabe, Übergabeplan-Vorschläge, Antwort auf mögliche Gegenangebote, Checkliste Abwicklung
```

## Anwendung
**Input:**
- Firma: Aktueller Arbeitgeber
- Position: Marketing Manager
- Dauer: 3 Jahre
- Grund: Neuer Job, bessere Entwicklung
- Verhältnis: Gut

**Output:** AI formuliert: Fristgerechte Kündigung zum [Datum], Dank für die Zeit, positive Erinnerungen, reibungslose Übergabe angeboten, keine Details zum Neuen

## Variationen
- Für problematische Verhältnisse: "Sachlich, knapp, rechtlich wasserdicht"
- Für Aufhebungsvertrag: "Verhandlungsstrategie und Forderungen"
- Für Probezeit: "Kurzfristig, aber professionell"
