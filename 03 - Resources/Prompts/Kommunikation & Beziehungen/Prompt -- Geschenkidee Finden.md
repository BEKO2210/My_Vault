# Geschenkidee finden

## Prompt

```
Rolle: Geschenkeberater mit Empathie
Kontext: Ich suche ein Geschenk für [PERSON]
Anlass: [GEBURTSTAG / WEIHNACHTEN / BESONDERER TAG]
Budget: [BETRAG]
Was ich über die Person weiß:
- Interessen: [LISTE]
- Hat schon: [WAS SOLLTE ICH NICHT SCHENKEN?]
- Persönlichkeit: [EHER PRAKTISCH / SENTIMENTAL / EXPERIMENTIERFREUDIG]
Aufgabe: Finde das passende Geschenk
Ausgabe:
1. 3 konkrete Ideen (mit Preis und Bezugsquelle)
2. DIY-Alternative (persönlich, zeitaufwändig)
3. Erlebnis-Geschenk (Erinnerung statt Ding)
4. Verpackungs-/Übergabe-Idee
```

## Beispiel

**Input:** Papa, 60. Geburtstag, 100€, Garten, Whisky, praktisch

**Output:** 
- Ideen: Hochwertige Gartenschere, Whisky-Tasting, Smart Garden
- DIY: Fotoalbum mit Familiengeschichten
- Erlebnis: gemeinsamer Grillkurs

## Variationen

- Für Partner: "Romantisch + persönlich"
- Für Kollegen: "Professionell, nicht zu privat"
- Für Kinder: "Bildung + Spaß"
